# Crypto NFT Game

Idle/Farming P2E game với NFT đại diện cho nhân vật/tài sản.

## Cấu trúc thư mục

- `contracts/` — Smart contract Solidity (NFT, token, logic farm)
- `indexer/` — Đồng bộ sự kiện on-chain vào database
- `backend/` — API server
- `frontend/` — Web app / game client
- `shared/` — ABI & type dùng chung
- `assets/` — Ảnh và metadata NFT
- `infra/` — Docker, CI/CD
- `docs/` — Tài liệu thiết kế game, tokenomics

## Trạng thái

Đang ở giai đoạn khởi tạo khung thư mục. Chưa có contract, chưa deploy testnet.
